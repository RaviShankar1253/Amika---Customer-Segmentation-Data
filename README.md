# Amika---Customer-Segmentation-Data
Cusutomer Purchase behavior analysis based on cohorts built using RFM model.
The project is focused towards understanding the product basket mix for different customer cohorts who are segmented based on RFM Model which stands for Recency, Frequency and Monetary value of the purchases over a year.
This analysis helped understand which product mixes are driving highest repeat rate with lowest lag times and which products are driving highest amount of new purchases helping to plan marketing strategy to directly focus towards improving customer retention and acquisition roadmap
